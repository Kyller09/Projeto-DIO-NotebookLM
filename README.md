# Fundamentos de Educação Financeira
Este repositório reúne notas introdutórias sobre educação financeira,
investimentos e conceitos econômicos básicos.

## Objetivo

Este repositório tem como objetivo organizar conceitos básicos de finanças
pessoais de forma simples e estruturada.

O material foi criado a partir de estudos pessoais e notas organizadas
com auxílio do NotebookLM.

## Conteúdos

- Fundamentos de educação financeira
- Orçamento pessoal
- Juros simples e juros compostos
- Reserva de emergência
- Introdução aos investimentos
- Inflação
- Dívidas e crédito

- ## Estrutura
Cada pasta contém notas em Markdown sobre um tema específico.

## Fontes

- Pai Rico Pai Pobre
- artigos de educação financeira
- relatórios econômicos
